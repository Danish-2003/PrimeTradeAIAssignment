Trader Performance vs Market Sentiment

Project Overview

This project analyzes how Bitcoin market sentiment (Fear vs Greed) relates to trader behavior and performance on Hyperliquid.

The notebook performs data alignment, feature engineering, behavioral segmentation, and exploratory modeling to evaluate whether sentiment acts as a market regime indicator for trader profitability.

How to Run
1️. Clone the repository
git clone <your-repo-link>
cd Trader-Sentiment-Analysis
2️. Install dependencies
pip install -r requirements.txt
3️. Open the notebook
jupyter notebook

Open:

Trader_Sentiment_Analysis.ipynb

Run all cells sequentially.
